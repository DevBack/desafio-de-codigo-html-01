O Que São?

Voltar

Tags HTML são instruções que informam ao navegador como formatar o texto. Você pode usar tags para definir itálico, quebras de linha, objetos, bullet points e muito mais. Essas tags estão presentes no HTML (Hypertext Markup Language) de todas as páginas da internet.

Os servidores leem o código HTML para entender e renderizar o conteúdo corretamente. Essa leitura é feita de cima para baixo, da mesma forma que você está lendo este artigo agora.

p - Parágrafo

Essa tag formata qualquer texto entre a tag de abertura e a tag de fechamento como um parágrafo padrão ou corpo de texto principal.

mark - Destaque ou Marcação

Essa tag formata qualquer texto entre a tag de abertura e a tag de fechamento com um background amarelo semelhante ao marca texto. O objetivo dessa tag é dar destaque a uma parte do texto.

Exemplo:
Marcação

small - Diminui o Texto

Essa tag formata qualquer texto entre a tag de abertura e a tag de fechamento com textos menores (como direitos autorais e outros comentários secundários).

i - Itálico

Essa tag formata qualquer texto entre a tag de abertura e a tag de fechamento como itálico.

u - Sublinhado

Essa tag formata qualquer texto entre a tag de abertura e a tag de fechamento como sublinhado.

strong - Importante

Essa tag formata qualquer texto entre a tag de abertura e a tag de fechamento como importante com estilo negrito.

ol - Lista Ordenada

Essa tag formata qualquer item de lista entre a tag de abertura e a tag de fechamento como uma lista ordenada.

Exemplo:

  1. item 1
  2. item 2
  3. item 3

ul - Lista Desordenada

Essa tag formata qualquer item de lista entre a tag de abertura e a tag de fechamento como uma lista desordenada.

Exemplo:

  • item 1
  • item 2
  • item 3

li - Item Lista

Essa tag formata qualquer texto entre a tag de abertura e a tag de fechamento como um item de lista.

Exemplo:

  • item 1
  • item 2
  • item 3

a - Hiperlink

A tag define um hiperlink, que é usado para vincular de uma página a outra. O atributo mais importante do elemento é o href, que indica o destino do link.

Exemplo:
Visit W3Schools.com!

hr - Separador

A tag define uma quebra temática em uma página HTML (por exemplo, uma mudança de tópico).

sub - Subscrito

A tag define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e às vezes é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H 2 O.

Exemplo:
H2O

sup - Sobrescrito

A tag define texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e às vezes é renderizado em uma fonte menor. Texto sobrescrito pode ser usado para notas de rodapé.

Exemplo:
Cubotv

blockquote - Citação

A tag especifica uma seção citada de outra fonte.

Exemplo:

For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.
